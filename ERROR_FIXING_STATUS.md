## Key Error Categories

### 1. Missing Getter/Setter Methods (Most Common)
- Affects DTO classes heavily
- Result of Lombok removal
- Examples: TimeSlot, FlowExecutionData, HeatmapStatistics DTOs

### 2. Type Mismatches
- UUID vs String conversions
- List<UUID> vs List<String> incompatibilities
- Object to specific type conversions

### 3. Ambiguous References
- Session type conflict (jakarta.jms.Session vs com.jcraft.jsch.Session)
- Needs explicit import or fully qualified names

### 4. Enum Switch Issues
- Switch cases must use unqualified enum names
- Common in adapter type handling

### 5. Missing Builder Patterns
- Many DTOs missing builder implementations
- Previously generated by Lombok
